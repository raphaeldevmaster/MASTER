Upper Case

Capital letter - Letter larger than its lower case. It is usually used at the beginning of phrases or names. It is also widely used to emphasize some text, word or phrase. An example of a capital letter is "A".

Sentence case

This function capitalizes the first letter of each sentence. A sentence starts right after a period "." For example, the phrase 'uppercase is bigger. lowercase is smaller', after using the function, it will be equal to 'Uppercase is bigger. Lowercase is smaller'.

Capitalized Case

This function capitalizes the first letter of each word. For example, the phrase 'all capital letters', after using the function, will be the same as 'All Capital Letters'.
